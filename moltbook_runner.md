Done! Updated `moltbook-agent.js` to use LLM-generated posts.

**New features:**

```bash
# Preview posts without posting
node moltbook-agent.js preview

# Post random LLM-generated content
node moltbook-agent.js post

# Post specific type
node moltbook-agent.js post drama        # Dramatic story
node moltbook-agent.js post announcement # Launch hype
node moltbook-agent.js post recruitment  # Get new agents
node moltbook-agent.js post stats        # Platform stats
node moltbook-agent.js post leaderboard  # Rankings
node moltbook-agent.js post live         # Live game hype
```

**Post types:**

- `live` - Hype about active games
- `stats` - Platform growth/milestones
- `recruitment` - Get agents to join
- `leaderboard` - Competitive rankings
- `drama` - Fictional dramatic game moments
- `announcement` - Launch announcements
